# Dominguez Marble - Natural Stone Contractor Website

A modern, responsive website for a general contractor specializing in natural stone work.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Sections Included**:
  - Hero section with call-to-action buttons
  - Services showcase (6 key services)
  - Portfolio gallery
  - About section with company stats
  - Contact form with validation
  - Footer with quick links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Inter)

## Getting Started

1. **Clone or download** this repository
2. **Open** `index.html` in your web browser
3. That's it! No build process or dependencies required.

## File Structure

```
DominguezMarble/

```

## Customization

### Colors
Edit the CSS variables in `styles.css` to match your brand:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #c9a961;
    --accent-color: #8b7355;
    /* ... */
}
```

### Content
- Update company information in `index.html`
- Replace placeholder portfolio images with actual project photos
- Modify services, contact information, and other content as needed

### Contact Form
The contact form currently shows an alert on submission. To make it functional:
1. Set up a backend service (e.g., Formspree, EmailJS, or your own server)
2. Update the form submission handler in `script.js`
